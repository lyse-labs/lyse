# create-lyse

One-command setup for Lyse.

```bash
npm create lyse@latest
```

This package is a thin wrapper that invokes `npx @lyse-labs/lyse init --first-run`. The real logic lives in the [`lyse`](https://www.npmjs.com/package/lyse) package.
